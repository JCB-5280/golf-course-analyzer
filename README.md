# Golf Course Analyzer (GCA)

This repository contains the strategic and product documentation for the Golf Course Analyzer project.

## Repository Structure

- `project-artifacts/`: Contains project-level documentation.
    - `project-charter/`: Individual components of the project charter (Mission, Vision, Objectives, etc.).
    - `presentations/`: Slide decks and presentation materials.
- `product-artifacts/`: Contains product-specific documentation and research.
    - `discovery/`: Market research, competitive analysis, and requirement sets.
    - `product_vision.md`: The overarching product vision document.
    - `interview_questions.md`: User and stakeholder interview guides.

## About GCA

The Golf Course Analyzer is a cloud-native platform designed to unify golf course operations through AI, IoT, and remote sensing, driving profitability and sustainability in the modern golf industry.
